# Atrativos Turísticos de Guarujá

Este projeto apresenta um catálogo visual dos principais **atrativos turísticos de Guarujá**, com destaque para mirantes, praias e pontos históricos da cidade.

## ✨ O que o site apresenta

- Cards com imagens e informações sobre cada atrativo
- Descrição dos locais e suas características
- Links externos para páginas com mais detalhes sobre cada ponto turístico
- Layout organizado com foco em navegação visual e conteúdo informativo

## 🛠️ Tecnologias empregadas

- **HTML5** para a estruturação das seções e conteúdo
- **CSS3** para ajustes de estilo personalizados
- **W3.CSS** para componentes, espaçamentos e responsividade
- **Imagens locais** em `assets/` para ilustrar os atrativos

## 👀 Como visualizar

### Opção 1: abrir diretamente no navegador

1. Abra o arquivo `index.html` na raiz do projeto.
2. O navegador carregará a página automaticamente.

### Opção 2: usar um servidor local

Se preferir, você pode subir um servidor simples com:

```bash
python -m http.server
```

Depois acesse no navegador:

```text
http://localhost:8000
```

## 📁 Estrutura do projeto

- `index.html` — página principal do site
- `style.css` — estilos personalizados do projeto
- `assets/` — imagens dos atrativos turísticos

## 📝 Observação

O site foi montado como uma apresentação estática, então não há backend nem dependências externas além do uso do [W3.CSS](https://www.w3schools.com/w3css/5/w3.css).
